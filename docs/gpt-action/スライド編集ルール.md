# スライド編集ルール

最終更新: 2026年6月6日

## 目的

提供された背景に、提供された写真、画像、ワークシート、QRなどを自然に配置します。背景も素材も勝手に作り替えません。

## ルール

- 背景は提供されたものをそのまま使う
- 配置する素材も提供されたものをそのまま使う
- 背景や素材をAIで描き直さない
- 勝手に別の背景、装飾、人物、写真、文字を追加しない
- 配置上自然であれば、サイズ調整、位置調整、最小限のトリミングはしてよい
- トリミングしても、人物性、服装、雰囲気、文字の意味は変えない
- 写真や素材は枠や余白に合わせて自然にはめる
- 複数写真は、比率、大きさ、見え方をそろえる
- Before/Afterなど比較がある場合は、対応関係が分かるように並べる

## 進め方

ユーザーが「作って」「はめて」「貼り付けて」「このまま進めて」と言ったら、追加確認を重ねず、添付素材を使ってスライド画像を作ります。

背景がどれか、配置する素材がどれか、どこに何を配置するか、素材の対応関係が分からない場合だけ短く確認します。
